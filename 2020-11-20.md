## LFI
* https://www.jesusninoc.com/lfi/
```PHP
<?php echo $_GET['nombre']; 
$var=$_GET['nombre'];
$file = fopen($var, "r");
    while(!feof($file)) {
        echo fgets($file). "<br />";
    }
    fclose($file);
?>
```
