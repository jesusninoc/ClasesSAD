# Detectar XSS
```PowerShell
# Obtener el código de la web https://github.com/jesusninoc/Seguridad/tree/master/GetPost

$web = iwr "http://localhost/seguridad/GetPost/"
$web | Get-Member -MemberType Properties
$web.Links | %{
    if($_.outerText -match ".php")
    {
        $_.outerText
    }
}
```
